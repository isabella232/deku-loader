# deku-copy-button

A copy button component for Deku.

![Demo](https://cldup.com/Q6mO30NAOh.png)

## Usage

```js
import CopyButton from 'deku-copy-button';

function render() {
  return <CopyButton label="My Copy Button" text="Some text" />
}
```
